# Dot files

## Installation

```
git clone git://github.com/andythigpen/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
# or install specific packages like:
# ./install.sh git tmux vim
```

## Uninstallation

```
cd ~/.dotfiles
./uninstall.sh
# or uninstall specific packages like:
# ./uninstall.sh git tmux vim
```
